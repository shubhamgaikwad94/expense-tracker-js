# Expense Tracker (JavaScript)

A simple expense tracker web application built using **HTML, CSS, and JavaScript**.

## Features
- Add income and expense transactions
- Automatically calculates balance, income, and expenses
- Delete individual transactions
- Data stored using browser localStorage
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
- Local Storage

## How It Works
- User enters transaction name and amount
- Positive amount → Income
- Negative amount → Expense
- Balance updates automatically
